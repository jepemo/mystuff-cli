Getting Started

To show your GitHub pinned repositories, add your username to ~/.mystuff/config.yaml:

generate:
  web:
    output: "~/mystuff_web"
    title: "{{ title }}"
    description: "{{ description }}"
    author: "{{ author }}"
    github_username: "your-github-username"
    menu_items:
      - name: "GitHub"
        url: "https://github.com/your-username"
      - name: "Blog"
        url: "/blog"

Then regenerate your site:

mystuff generate web
