Getting Started

To customize this site, edit your ~/.mystuff/config.yaml:

generate:
  web:
    output: "~/mystuff_web"
    title: "{{ title }}"
    description: "{{ description }}"
    author: "{{ author }}"
    menu_items:
      - name: "Home"
        url: "/"
      - name: "GitHub"
        url: "https://github.com/{{ author }}"
      - name: "Blog"
        url: "/blog"

Then regenerate your site:

mystuff generate web
